Para rodar a aplicação usei alguns módulos em npm, onde importei de bibliotecas do próprio VSCode, sendo elas, Node, TypeScrypt, Server, Babel core e React App, onde estão todas listadas nos arquivos caso de alguma incompatibilidade. Foi Utilizado para a criação do backend, o VSCode como IDE, MongoDB para banco de dados, e linguagens como typeweb, onde inclui js, css e html. E do front end, ferramentas como yarn, para iniciar e testar o projeto, e ReactJS, integrando com os dados do backend e apresentando em tela pro usuário final.
